Встроенные типы и операторы

Типы данных

К стандартным типам данных в Python относят:
  • Числа (Numbers)
  • Строка (String)
  • Список (List)
  • Кортеж (Tuple)
  • Словарь (Dictionary)
  • Сет (Set)

Операторы

Арифметические операторы в Python:
  • Сложение -> +
  • Умножение -> *
  • Вычитание -> -
  • Деление -> /
  • Деление по модулю -> %
  • Возведение в степень -> **
  • Целочисленное деление -> //
