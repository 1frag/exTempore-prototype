Установка и запуск

Windows

Для установки интерпретатора Python на ваш компьютер, первое, что нужно сделать – это скачать дистрибутив. Загрузить его можно с официального сайта, перейдя по ссылке https://www.python.org/downloads/

Для операционной системы Windows дистрибутив распространяется либо в виде исполняемого файла (с расширением exe), либо в виде архивного файла (с расширением zip). Необходимо только скачать исполняемый файл и следовать инструкциям установки.

ВАЖНО: при установке отметьте пункт "Add Python to environment variables", чтобы иметь возможность запускать интерпретатор из командной строки, а также установите пакетный менеджер pip.

Ubuntu

Чаще всего интерпретатор Python уже в входит в состав дистрибутива. Это можно проверить набрав в терминале:

python3

Если у вас, при попытке запустить Python, выдается сообщение о том, что он не установлен, то установить его можно командой:

sudo apt-get install python3
